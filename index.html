Q: What is a variable?

A: A variable is a way to hold a value you assign to it. This can be a simple text or number, but also a function or object.

Q: What is the difference between let and const?

A: const is a constant variable, meaning you assign it once and you can't reassign it. let is a variable which can be reassisgned.

Q: What is a method?

A: A method is an action that can be performed on objects. A method is a property containing a function.

Q: What is a object?

A: An object is a variable containing multiple values written as name:value pairs which are called properties.

Q: What is a function?

A: A function is a block of code designed to perform a specific task.This function is invoked when "something" invokes it.
